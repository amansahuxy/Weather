
# Weather App 🌦️

A simple and responsive Weather App user interface built using HTML and CSS. The application provides a clean design for displaying weather information such as temperature, city name, humidity, and wind speed.

## Features

* Search bar for entering city names
* Weather icon display
* Temperature display
* City name display
* Humidity information
* Wind speed information
* Responsive card-based layout
* Modern gradient background design

## Technologies Used

* HTML5
* CSS3

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder:

```bash
cd weather-app
```

3. Open `index.html` in your browser.
